Upon shedding their previous corporate parent, Bayou Steel Group was looking for a full refresh of their brand and website. This build included lots of imagery, SVG iconography, videos, and more to showcase the company and its workers—all in a fast-loading, responsive package.Custom Google Map API integration was used extensively to showcase the different locations as well as differintiate the types with iconography. This build was launched on an EC2 instance and deployed with Git using Trellis and Bedrock for server parity.

Overview

  • WordPress
  • Custom Google Map API integrations
  • Parallax
  • Background video
  • Isotope image collages
  • Custom grid system
  • Custom accordion menus
  • Trellis/Bedrock
  • Sass
  • Gulp
  • Fully responsive and content managed
