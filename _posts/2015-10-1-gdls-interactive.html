General Dynamics Land Systems wanted an update to their interactive touch displays for trade shows and conferences. Updating from the old Flash-based interactive, this version was built from the ground up with heavy use of SVGs, CSS transitions, transforms, and filters. The build maintains modularity utilizing the power of Angular to easily update the content for future conferences.

The interactive is optimized for a 1920x1080 resolution

Overview

  • AngularJS
  • CSS Animations
  • CSS Transforms and Filters
  • Touch Optimized Swipe Navigation
  • SVGs
  • Sass
  • Gulp

View Experience (requires browser resolution set to 1920x1080)
