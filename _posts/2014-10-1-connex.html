The HealthPlus Connex service microsite was built for training purposes for HealthPlus providers. Using the look and feel of the the main site to create UI components. The Connex service could not be used with Internet Explorer, so conditional stylesheets and IE 10-11 specific media queries were utilized to prompt the user to use a different browser.
